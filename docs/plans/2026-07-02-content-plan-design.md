# 内容规划中心 — 设计文档

## 概述

为「墨笔 AI」新增内容规划中心（`/content-plan`），将公众号内容生产从「单篇写作」升级为「选题 → 排期 → 写作 → 发布」的全管线管理，核心解决「不知道写什么能火」的痛点。

## 页面布局

整体分三大垂直区域：

### 1. 灵感中心（顶部 ~40% 视口）

AI 选题推荐引擎，每次加载生成 5 个选题。

**选题卡片包含：**
- 热度指数（1-100，条形或数字展示）
- 标题
- 「为什么能火」分析（热点关联 + 账号定位切入角度）
- 关联热点标签
- 「纳入日历」按钮

**操作区：**
- 「换一批」：重新 AI 生成 5 个选题
- 「自定义灵感」：弹出输入框，用户输入方向，AI 生成 3 个切入角度

**纳日流程：** 点击「纳入日历」→ 弹出表单（标题可修改 + 选择发布日期）→ 存入选题池（状态：灵感）

### 2. 选题池（中部）

展示所有已保存的选题，支持筛选和状态管理。

**筛选栏：** 全部 / 灵感 / 写作中 / 待发布 / 已发布

**每项展示：** 选题标题 | 状态标签 | 截止日期（如有）

**操作：** 编辑 | 跳转写文章 | 删除 | 改状态

**底栏：** 「+ 手动添加选题」（无 AI 辅助，直接输入标题和日期）

### 3. 内容日历（底部，月视图）

月视图日历，有内容的日期以小圆点标记。点击日期显示当天选题/文章列表。

**今日焦点卡片：** 显示今天的写作任务（状态为「写作中」或「待发布」的选题），提供「一键写作」按钮（跳转到文章生成页并带入选题）。

## 状态流转

```
灵感 ──→ 写作中 ──→ 待发布 ──→ 已发布
  ↑         ↑           ↓
  └──── 可回退任何状态 ──┘
```

## 数据模型

```typescript
interface ContentTopic {
  id: string;
  title: string;
  status: '灵感' | '写作中' | '待发布' | '已发布';
  publishDate: string | null;       // YYYY-MM-DD，选填
  createdAt: string;
  heat: number | null;              // 灵感来源时 AI 给的热度
  reason: string | null;            // 为什么能火分析
  tags: string[];
  source: 'ai' | 'manual';          // AI推荐 or 手动添加
}
```

localStorage key: `mobi-content-plan`

侧边栏新增入口「内容规划」，图标用 `Calendar`，排到热点追踪下方。

## 依赖

- 读取 `settings` 中的账号定位（已有）
- 读取热点数据（已有 `hotTopics` 逻辑）
- 写文章时回写选题状态（文章生成后自动将选题状态改为「已发布」）
- 与文章生成页的联动：从内容日历点「一键写作」携带选题标题到 `/article-generation?topic=xxx`
