# 公众号排版美化升级 — 设计文档

## 概述

对现有「公众号排版」页面进行增量升级，在保持现有 Markdown 编辑 + 手机预览结构的基础上，叠加三大功能：模板库、样式微调、排版组件。

## 方案

采用方案 2（增量升级），不改动现有页面架构，3 天内可上线。

---

## 1. 模板库

### 当前状态
6 个主题：翡翠青、优雅蓝、炫彩紫、暖阳橙、玫瑰红、极客黑

### 升级目标
扩充至 15 套精品模板，按分类展示：

| 分类 | 模板 |
|------|------|
| 🔥 热门推荐 | 极简白、温暖米灰、墨绿文艺 |
| 🎨 简约风 | 清水、素笺、留白 |
| 📰 杂志风 | 封面头条、双栏、图文混排 |
| ✨ 文艺风 | 手账、胶片、诗篇 |
| 💼 商务风 | 报告、新闻简报、观点文 |
| 🎉 活泼风 | 糖果、渐变、卡片 |

### 交互
- 分类标签筛选 + 模板卡片网格展示
- 点击模板 → 预览区立即刷新
- 每个模板内置：name, category, colors (primary, title, text, bg, accent), typography (titleSize, bodySize, lineHeight, spacing)

---

## 2. 样式微调面板

### 位置
模板选择器下方，可折叠展开

### 可调参数
- 主色（预设色板 8 色 + 自定义 hex）
- 标题色
- 背景色
- 正文字号（小/中/大）
- 行距（1.6 / 1.8 / 2.0 / 2.2）
- 段间距（12px / 16px / 20px）

### 原则
- 默认折叠，不干扰主操作流
- 每项调整实时更新预览
- 「重置为模板默认」一键恢复

---

## 3. 排版组件库

### 新增 6 个排版组件

| 组件 | Markdown 语法 | 预览效果 |
|------|-------------|---------|
| 💡 提示框 | `:::tip` 内容 `:::` | 带背景色和图标的提示区块 |
| 📦 引用卡片 | `:::card` 内容 `:::` | 圆角卡片，带阴影边框 |
| 🌟 金句 | `:::quote` 内容 `:::` | 大字加粗，左侧竖线装饰 |
| 📊 对比 | `:::compare` A vs B `:::` | 左右分栏对比 |
| ❓ Q&A | `:::qa` Q: xxx A: xxx `:::` | 问答样式，Q 和 A 不同色 |
| 🎯 要点 | `:::point` 内容 `:::` | 带圆点图标的要点卡片 |

### 交互
- 工具栏新增一组组件按钮
- 点击后 Markdown 编辑器光标处插入对应片段 + 占位文字
- 一键排版时组件语法渲染为对应 HTML 样式
- 复制到公众号编辑器后样式保留

## 实现计划

1. 更新模板数据（themes 数组从 6 个扩展到 15 个，添加分类、元数据）
2. 重构模板选择器（分类标签 + 卡片网格）
3. 添加样式微调面板（折叠 + 实时预览联动）
4. 添加 6 个排版组件的工具栏按钮
5. 更新 mdToHtml 函数，支持 `:::` 自定义语法解析
6. 测试验证所有模板和组件效果
